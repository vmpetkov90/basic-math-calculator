## 🚀 Features

- Basic arithmetic operations: **add, subtract, multiply, divide**
- Decimal support with validation
- Prevents invalid inputs (multiple decimals, missing numbers)
- Error messages for:
  - Invalid input
  - Missing operands
  - Division by zero
- Reset and delete functionality
- Clean, Casio‑inspired UI
- Fully responsive layout

---

## 🛠️ Technologies Used

- **HTML5** – structure  
- **CSS3** – layout, grid, styling, responsive design  
- **JavaScript (ES6)** – calculator logic, state handling  
- **Google Fonts / System Fonts**

---

## 🧮 How It Works

- User inputs numbers via on‑screen buttons  
- Operators store the first number and selected operation  
- The calculator validates input before performing calculations  
- Results are displayed in the format:  
  `number1 operator number2 = result`  
- After a calculation, typing a new number resets the calculator automatically
